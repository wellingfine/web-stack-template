# web-stack-template
多技术栈的web建站模板，快速构建项目站点。包括以下模板：

* webpack-react-koa
* webpack-react-eggjs 
* webpack-vue-eggjs
* webpack-vue-koa


# 特性

* 本地开发
* node+web热重载
* 资源定位
* 服务端渲染
* 

# 规范约定

* 所有目录名称使用单数形式
* 目录名小写,一般文件名小写,类名大写
* tab缩进
* 变量命名驼峰,类名首写字母大写
